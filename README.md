Hangman Game

Introduction

Welcome to Hangman, a classic word-guessing game implemented in Python. The game randomly selects a word from a predefined list and allows the player to guess the word by suggesting letters. This game is based on the traditional Hangman game, with a unique implementation in Python.

How to Play
The game starts by displaying a blank word representation, replacing each letter with an underscore.
The player is prompted to enter a single letter.
If the letter is in the word, it is revealed in the correct position.
If the letter is not in the word, the game displays a hangman graphic, and the player loses a life.
The game continues until the player either guesses the word correctly or runs out of lives.

Game Features
Randomly select a word from a list of 25 words related to sports and fitness.
Displays a hangman graphic that changes as the player loses lives.
Tracks the player's lives and displays the remaining lives.
Allows the player to guess the word by suggesting letters.

Technical Details
Written in Python 3.x.
Uses a list of predefined words and a list of hangman graphics.
Implements a simple game loop that continues until the game is won or lost.

Inspiration
This game is a re-implementation of the classic Hangman game, with a unique codebase. The idea of Hangman is not new, but the code and implementation are original.

Running the Game
To run the game, simply execute the Python script in your terminal or command prompt.
